КАЛЕНДАРНЫЙ ПЛАН ВЫПОЛНЕНИЯ НИОКР
Проект: AIDA-T (Agrobotic Intelligent Data Analyzer for Tomatoes)
Автономная робототехническая система интеллектуального мониторинга и диагностики томатов в промышленных теплицах

Общий срок выполнения НИОКР: 12 месяцев
Заявитель: Давиденко Сергей Александрович
Конкурс: Старт-ИИ-1-2025 (очередь 1)

ЭТАП 1. АППАРАТНО-МЕХАНИЧЕСКАЯ РАЗРАБОТКА И ИНТЕГРАЦИЯ КОМПОНЕНТОВ
Сроки выполнения: месяцы 1-6 (6 месяцев)
Финансирование: 50% от общей суммы гранта

Содержание работ:

1.1. Проектирование и изготовление рельсовой системы (месяцы 1-2)
- Разработка конструкции рельсовых путей для движения между рядами томатов
- Проектирование нейлоновых рельсовых колес с оптимальным профилем
- Расчет нагрузок и выбор материалов для рельсовой системы
- Изготовление экспериментальных образцов рельсовых компонентов

1.2. Установка камер и доработка ходовой части (месяцы 2-3)
- Монтаж промышленных Ethernet-камер на камерную мачту
- Интеграция меканум-колес для движения по бетонному покрытию
- Доработка системы переключения между рельсовым и колесным режимами
- Установка бесщеточных мотор-редукторов с требуемым пусковым моментом

1.3. Герметизация электронных компонентов (месяцы 3-4)
- Обеспечение пылевлагозащиты IP65 для работы в условиях теплиц
- Герметизация блоков управления и вычислительной платформы
- Защита электрических соединений от повышенной влажности
- Тестирование герметичности в климатических условиях теплицы

1.4. Разработка middleware для межкомпонентного взаимодействия (месяцы 4-5)
- Создание программного слоя для взаимодействия между аппаратными модулями
- Разработка драйверов для управления мотор-редукторами и сенсорами
- Реализация протоколов обмена данными между камерами и вычислительным блоком
- Интеграция системы активной амортизации с модулем управления

1.5. Низкоуровневые тесты аппаратных компонентов (месяцы 5-6)
- Тестирование работоспособности гибридной ходовой системы
- Проверка точности позиционирования камерной мачты
- Тестирование системы переключения между режимами движения
- Проверка энергопотребления и автономности работы

1.6. Настройка сенсорного setup и калибровка системы (месяцы 5-6)
- Калибровка промышленных Ethernet-камер для условий теплицы
- Настройка параметров съемки (экспозиция, баланс белого, фокус)
- Синхронизация работы нескольких камер для многоракурсной съемки
- Тестирование качества получаемых изображений в различных условиях освещения

1.7. Реализация слияния кадров в единое изображение (месяц 6)
- Разработка алгоритмов сшивки (stitching) кадров с пересекающимися углами обзора
- Оптимизация качества итогового изображения
- Устранение артефактов на границах сшивания
- Валидация точности геометрических измерений после слияния кадров

1.8. Подача заявки на полезную модель робототехнического комплекса (месяц 6)
- Подготовка описания полезной модели гибридной ходовой системы
- Оформление чертежей и технических схем для заявки
- Подача заявки в Роспатент на полезную модель "Мобильная платформа с гибридной ходовой системой"

Результаты этапа 1:
- Функционирующая мобильная платформа с гибридной ходовой системой
- Калиброванная система технического зрения с точностью позиционирования ±1.5 мм
- Алгоритмы слияния кадров для повышения качества изображений
- Программный middleware для управления аппаратными компонентами
- Заявка на полезную модель робототехнического комплекса
- Промежуточный научно-технический отчет

ЭТАП 2. РАЗРАБОТКА ПРОГРАММНОГО ОБЕСПЕЧЕНИЯ И КОМПЛЕКСНЫЕ ИСПЫТАНИЯ
Сроки выполнения: месяцы 7-12 (6 месяцев)
Финансирование: 50% от общей суммы гранта

Содержание работ:

2.1. Написание верхнеуровневой архитектуры системы (месяцы 7-8)
- Разработка модульной архитектуры на базе ROS для интеграции всех подсистем
- Создание системы автономной навигации и планирования маршрутов
- Реализация алгоритмов принятия решений для диагностики растений
- Разработка системы управления энергопотреблением для 8+ часов автономной работы

2.2. Написание графического пользовательского интерфейса (месяцы 8-9)
- Создание веб-интерфейса для мониторинга работы системы AIDA-T
- Разработка модулей визуализации результатов диагностики заболеваний
- Реализация системы отчетности с картами состояния посадок
- Создание интерфейса для настройки параметров работы системы

2.3. Автоматизация низкоуровневых и верхнеуровневых тестов (месяцы 9-10)
- Разработка автоматизированных тестов для проверки аппаратных компонентов
- Создание тестового фреймворка для валидации алгоритмов компьютерного зрения
- Реализация системы непрерывного тестирования (CI/CD) для программных модулей
- Автоматизация тестирования CNN-модели на контрольных изображениях

2.4. Написание технической документации и инструкций (месяц 10)
- Подготовка руководства пользователя для агрономов и операторов
- Создание технической документации по эксплуатации и обслуживанию
- Разработка инструкций по калибровке и настройке системы
- Оформление программной документации по алгоритмам ИИ

2.5. Испытания на вибро- и климатическую устойчивость (месяц 10)
- Тестирование работоспособности системы при вибрациях во время движения
- Проверка функционирования в условиях повышенной влажности теплицы (85-95%)
- Испытания при температурных перепадах (15-35°C)
- Валидация стабильности работы электронных компонентов

2.6. Экспериментальные тесты алгоритмов ИИ (месяцы 10-11)
- Тестирование CNN-модели для диагностики мучнистой росы на контрольных образцах
- Валидация алгоритмов RANSAC и PointNet для оценки объема плодов
- Проверка точности диагностики в различных условиях освещения
- Оптимизация параметров нейронных сетей для достижения точности 86-87%

2.7. Полевые тесты в условиях промышленной теплицы (месяцы 11-12)
- Проведение испытаний системы AIDA-T в реальных условиях тепличного комплекса
- Тестирование автономной навигации между рядами томатов
- Валидация точности диагностики заболеваний на живых растениях
- Проверка точности оценки объема реальных плодов (погрешность ≤10%)
- Тестирование круглосуточной работы системы в течение 8+ часов

2.8. Регистрация программ для ЭВМ (месяц 12)
- Подготовка документации для регистрации программы диагностики заболеваний
- Оформление заявки на регистрацию алгоритмов оценки урожайности
- Подача заявок в Роспатент на программы для ЭВМ
- Подготовка описаний программных решений с указанием функциональности

Результаты этапа 2:
- Полнофункциональное программное обеспечение системы AIDA-T
- Графический интерфейс для мониторинга и управления
- Автоматизированная система тестирования всех компонентов
- Комплект технической документации и инструкций пользователя
- Подтвержденная работоспособность в условиях промышленной теплицы
- Валидированная точность диагностики мучнистой росы 86-87%
- Подтвержденная точность оценки объема плодов с погрешностью ≤10%
- Заявки на регистрацию программ для ЭВМ
- Заключительный научно-технический отчет
- Протоколы комплексных испытаний экспериментального образца

СВЯЗЬ С ТЕХНИЧЕСКИМ ЗАДАНИЕМ:

Этап 1 обеспечивает создание аппаратной платформы с характеристиками:
- Точность позиционирования ±1.5 мм на пиксель
- Гибридная ходовая система (рельсы + меканум-колеса)
- Пылевлагозащищенное исполнение для теплиц

Этап 2 обеспечивает создание программного обеспечения с характеристиками:
- Точность диагностики мучнистой росы 86-87%
- Точность оценки объема плодов с погрешностью ≤10%
- Автономная работа 8+ часов в сутки
